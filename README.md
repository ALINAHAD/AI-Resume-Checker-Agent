# ResumeFit AI – ATS Resume & JD Matcher

An AI-powered Resume Checker Agent that compares resumes against job descriptions and generates ATS-style feedback.

## Features
- Resume vs JD matching (0–100 score)
- Missing keywords & skills detection
- Improvement suggestions
- Telegram bot input
- Email report delivery

## Tech Stack
Make.com, Telegram Bot API, Make AI Agents, Data Store, Gmail Integration

## Workflow
Telegram → Make.com Trigger → Resume Extract → AI Analysis → Report → Telegram + Email
